# 會員點餐系統
## 登入與註冊功能修改老師的sayit專案，css評分星星使用cdnjs套件設計 其他為原創，主頁面使用 Document Object Model 進行操作

### 網頁功能
* 點餐
* 結帳
* 註冊
* 登入
* 登出

### 大致上的運作流程
註冊帳號後，資料庫儲存用戶資料後，接著直接登入，進入點餐頁面進行點餐和結帳

### 待補充功能
* 使登出後無法回到主頁面
* 主頁的分頁部分
